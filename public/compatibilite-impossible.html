Navigateur non supporté par l'application

La configuration de votre navigateur est trop ancienne pour utiliser l'application !

Vous utilisez actuellement la version #version# du navigateur #browser#

L'application est compatible à partir des versions suivantes :

  • IE11 (fonctionnement minimum requis)
  • Firefox version 53
  • Chrome version 51
  • Edge version 18
  • Opera version 38
  • iOs 10 (Safari / Chrome)
  • Android 4.4 (Chrome, Firefox, UC)
